Pseudocoding and Problem-solving

Challenge

The main challenge is to describe relatively complex tasks by steps in an easy to understand language. These tasks include daily life and computers. We are more inclined to use video to learn a new skill, and the text tutorial maybe strange to us. However, we decided to put it in words as perfect as possible.

Problems

We have a problem with the steps, because we don't know what materials and tools the person reading this instruction has. So we offer as many material choices as possible.

Sometimes we also don't know how to use correct words to express our imaginary actions properly. So we wrote the steps as detailed as possible, and some of the steps were split into two parts. We've also changed a lot of words so that we can find the words closest to our meaning.

Break complicated tasks down into sub-steps

Make Sushi:

  1. Steam some rice.
  2. Put some rice vinegar into the steamed rice.
  3. Cut the filling ingredients (Choose from the following: cucumber, carrot, ham, salmon, etc) into sticks.
  4. Put one piece of seaweed on a bamboo sushi rolling mat with plastic wrap.
  5. Put the rice with rice vinegar on the seaweed evenly
  6. Flatten the rice with a spoon.
  7. Put ingredient sticks on one side of the rice.
  8. Add mayonnaise or other condiments you like.
  9. Roll mat from that side.
  10. Squeeze this roll.
  11. Divide mat and sushi.
  12. Cut the sushi roll into small pieces with a knife.
  13. Enjoy.

Task 2: Pseudocode a Computer Task

A list of high-level computer tasks

  • Jigsaw Puzzle
  • Guess a Number Game
  • Math Calculator
  • Age Calculator
  • Analog Clock

Break complicated tasks down into sub-steps

Guess a Number Game:

  1. Create a random number between 1 and 100.
  2. Accept user input.
  3. Judge whether the number entered by the user is equal to the random number. If they are equal, the user wins the game. Else tell the user that the imputed number is smaller or bigger than the random number.
  4. Repeat 2-3 until the user win. ( You can also set a limit on the number of guess times).
